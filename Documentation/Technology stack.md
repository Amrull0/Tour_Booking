# 7. Обоснование выбора стека технологий

## 7.1. Язык программирования: Java
Язык программирования Java, так как он обладает рядом преимуществ:<br>
‒ кроссплатформенность благодаря технологии JVM, что позволяет запускать приложение на различных операционных системах;<br>
‒	развитая экосистема библиотек и фреймворков;<br>
‒	высокая производительность и стабильность работы;<br>
‒	широкие возможности для реализации клиент-серверной архитектуры.<br>

## 7.2. Фреймворки: Android SDK + Firebase
Android SDK предоставляет все необходимые компоненты для нативной разработки. Firebase (Firestore, Authentication) выбран для backend-части - предлагает готовые решения для базы данных, аутентификации и хостинга, не требуя написания серверного кода.

## 7.3. СУБД: Cloud Firestore
Облачная NoSQL база данных от Google. Позволяет хранить данные в режиме реального времени с автоматической синхронизацией между устройствами. Не требует администрирования серверов, масштабируется автоматически.

## 7.4. Инструменты тестирования: Android Studio + Firebase Test Lab
Android Studio включает все необходимые инструменты для отладки и тестирования. Firebase Test Lab предоставляет облачную инфраструктуру для тестирования на реальных устройствах, что обеспечивает проверку совместимости на разных моделях смартфонов.
