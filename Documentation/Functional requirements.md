### Модуль: Управление аккаунтами (ACC)

| Функция | ID | Описание | Доступ | Статус |
| :--- | :--- | :--- | :--- | :--- |
| **Регистрация нового пользователя** | ACC-001 | Пользователь создаёт новую учётную запись с логином, паролем и контактной информацией. | Все | Реализовано |
| **Авторизация в системе** | ACC-002 | Пользователь вводит логин и пароль для входа в приложение. | Все | Реализовано |
| **Просмотр профиля** | ACC-003 | Пользователь может просмотреть свои личные данные и историю бронирований. | Все | Реализовано |
| **Редактирование профиля** | ACC-004 | Пользователь может изменить свои контактные данные и пароль. | Все | Реализовано |
| **Управление пользователями** | ACC-005 | Администратор может просматривать, редактировать или блокировать учетные записи пользователей. | Администратор | Реализовано |
| **Удаление пользователя** | ACC-006 | Администратор удаляет учётную запись пользователя при необходимости. | Администратор | Реализовано |

### Модуль: Управление турами (TOU)

| Функция | ID | Описание | Доступ | Статус |
| :--- | :--- | :--- | :--- | :--- |
| **Просмотр списка туров** | TOU-001 | Пользователь может просматривать все доступные туры с фильтрацией по стране, дате и стоимости. | Все | Реализовано |
| **Просмотр деталей тура** | TOU-002 | Пользователь может открыть подробное описание выбранного тура, включая программу, даты и стоимость. | Все | Реализовано |
| **Добавление нового тура** | TOU-003 | Администратор добавляет новые туры, указывая название, дату, стоимость и описание. | Администратор | Реализовано |
| **Редактирование тура** | TOU-004 | Администратор может редактировать информацию о туре, включая даты, стоимость и описание. | Администратор | Реализовано |
| **Удаление тура** | TOU-005 | Администратор удаляет тур из базы данных, если он больше не актуален. | Администратор | Реализовано |
| **Установка дат тура** | TOU-006 | Администратор устанавливает диапазон дат тура. | Администратор | Реализовано |
